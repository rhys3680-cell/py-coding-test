# exp_000 — 파이썬 기본기 + 베이스라인

## 배경

- JS는 능숙, 파이썬은 거의 처음.
- 코테 사고력 자체는 어느 정도 있을 것으로 추정 (JS에서 이미 풀어본 경험 가정).
- 병목은 **"파이썬 문법/메서드를 몰라서 생기는 마찰"**일 가능성이 큼.

## 관찰 목표 (이 실험은 가설 검증보다 탐색/베이스라인 측정)

1. **JS → Python 전환 비용**이 어느 수준인지 시간/정답률 수치로 남긴다.
2. **파이썬 특화 마찰점** (JS에는 없는/다른 개념)을 Top N개 식별한다.
3. 이후 실험들이 개선 여부를 비교할 **베이스라인 지표**를 확보한다.

## 수집할 지표 (베이스라인)

- 하루 푼 문제 수
- 평균 소요시간 (Lv.0, Lv.1 각각)
- 퍼스트 트라이 정답률
- 해설 본 비율
- 주관적 등급 분포 (Again/Hard/Good/Easy)
- 파이썬 특화 마찰 발생 빈도 (자유 메모에서 추출)

## 비가설

- "빠르게 숙달된다/느리게 숙달된다"를 판정하지 않음. **수치를 일단 남기는 것**이 목적.

## 성공 기준

- 7일간 jsonl 로그 10건 이상 축적
- 회고에서 **파이썬 마찰점 Top 3** 식별
- Lv.2 진입 가능 여부에 대한 판단 근거 확보
