# 실험 루프

작은 단위 실험 → 데이터 수집 → 회고 → 검증된 것만 붙여나가기.

## 원칙

1. 한 실험에 **하나의 가설/관찰 목표**
2. **수치로 남긴다** (jsonl + markdown)
3. **검증 후** 다음 실험에 통합
4. 자동화는 **필요해진 시점**에만

## 실험 단위 구조

```
exp_NNN_이름/
├── hypothesis.md       # 뭘 검증/관찰하는가
├── protocol.md         # 어떻게 할 건가 (기간, 기록 방식)
├── data.jsonl          # 풀이 1건 = 1줄
├── retrospective.md    # 끝난 뒤 회고
└── (필요 시 보조 문서)
```
